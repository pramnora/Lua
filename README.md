## Simple programs...

### Hello, world!

> print("Hello, world!")  

### Add 2 numbers

> print(1+2)

### Variables

> greeting = "Hello, world!"  
> num = 3.14  
> print(greeting)  
> print(num)  
> --output...  
> --Hello, world!  
> --3.14  

**NOTE**: The double hypen...is how you create a single line comment in Lua.
